# kintone-dts-gen

[![npm version](https://badge.fury.io/js/%40kintone%2Fdts-gen.svg)](https://badge.fury.io/js/%40kintone%2Fdts-gen)

Type definition for kintone customize and

```
npm install -D @kintone/dts-gen
```
